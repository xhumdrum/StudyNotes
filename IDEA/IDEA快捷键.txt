1、 设置字体
	file-->setting-->输入font-->选择Font-->设置字体样式和字体大小

2、快速生成 main 方法
	psvm

3、快速生成输出语句
	sout

4、IDEA是自动保存，不需要按 ctrl+s

5、删除一行 
	ctrl + y

6、运行
	代码上右键-->run
	或者点击左侧的绿色箭头
	ctrl + shift + F10

7、左侧窗口中的列表展开关闭
	左箭头关闭
	右箭头展开
	上下箭头移动

8、退出任何窗口，按 esc

9、任何新增/新建/添加的快捷键
	alt + insert

10、代码窗口变大变小切换
	ctrl + shift +F12

11、代码窗口页面切换
	alt + 左箭头/右箭头

12、有数字标记的窗口打开关闭切换
	alt + 数字

13、查看方法的参数提示
	ctrl + p

14、注释
	单行注释 ctrl + \
	多行注释 ctrl + shift + \
